## Instrucciones de uso

1. Clonar el repositorio.
2. Ejecutar `npm install` para instalar dependencias.
3. Ejecutar `ng serve` para iniciar la aplicación en modo desarrollo.
4. Ejecutar `ng test` para correr las pruebas.
